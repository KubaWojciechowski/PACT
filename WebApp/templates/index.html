PACT is a computational pipeline for prediction of amyloid cross-interactions.

Guide:

PATH takes as an input csv file with the folowing columns: interactor,seq1,interactee,seq2 Where interactor and interactee are the names, and seq1 and seq2 are sequences of interacting proteins in fasta format
